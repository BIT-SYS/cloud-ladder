## 编译与测试

### Windows

*如果你安装了最新版 gradle，也可以用 `gradle` 代替以下命令。*

构建项目:

```bash
./gradlew.bat build
```

测试项目:

```bash
./gradlew.bat test
./gradlew.bat testASTListener

```

### *nix & MAC OS

*如果你安装了最新版 gradle，也可以用 `gradle` 代替以下命令。*

构建项目:

```bash
./gradlew build
```

测试项目:

```bash
./gradlew test
./gradlew testASTListener

```
